What are Priority Objectives?

These are the main outcomes you want ForSys to focus on when identifying areas for treatment — like source exposure, wildfire hazard potential, or total live tree carbon.

What happens if you select only one priority objective?

If you select 1 objective, ForSys will prioritize areas with the highest value for that single objective.

What happens if you select two priority objectives?

If you select 2 objectives, ForSys scores each stand relative to others for the selected metrics and combines those scores. Metrics with more stands with high relative values will have a stronger influence on the overall result.

Can I adjust the weighting for Priority Objectives?

By default, both objectives are equally weighted (1x:1x). You can adjust the weighting to give one objective more influence over the results (e.g., 2x:1x).
