Start planning scenarios to generate project areas.

Enter a name for the scenario at the top left, select the management question, enter the constraints and select Generate. Only one management question can be selected per run.

After project areas have been generated, select "+ New Treatment Plan" to begin Impacts planning. Treatment plans can be reviewed by selecting the Treatment Plans tab at the top left.

The scenario page allows you to create and view all the scenarios that you have created for a specific Planning Area, including treatment impacts.

For each scenario that has been started or completed, you will see the number of project areas and the status of each scenario. Click on the scenario summary to go into the scenario. Archived scenarios can also be accessed from this page.

In the left panel you’ll find the collaborative Notes panel, where all members of the planning team can leave important information regarding the Project. Notes are viewable to all users who have access to that Planning Area.

The treatment plan page shows all project areas and the number of stands with treatments applied per project area. Click on the project area on the map to begin treatments. To search on applied treatments, use the search box at the left. Selecting "View Treatment Layers" will display all stands with their applied treatments on the map. When planning is complete, select “Run Treatment Analysis” to generate the impacts analysis.

To apply treatments, select stands by clicking or dragging over an area. Selecting one/many stands will bring up the "Apply Treatment" popup, which allows selection from various treatment types for each stand. The pop-up also enables the selection of single treatments, or sequenced treatments.

View results of the direct impacts treatments by changing the available metrics, selecting different project areas, selecting different treatment types and viewing stand-level results by selecting a hexagon.

This page allows you to explore all the data in the RRKs for your region. You can select a region from the dropdown menu, and then use the selectors below to set up your map(s). You can look at 1, 2 or 4 maps simultaneously, and select a different set of data and layers to visualize for each map. The map area remains constant for all maps.
